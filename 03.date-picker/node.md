## 启动

```bash
# 默认会找 App.vue 文件
# vue serve App.vue
vue serve
```



## 安装环境

```bash
# 富于表现力、动态的、健壮的 CSS
# -S 即--save（保存）
# -D 即--dev（开发）
# 最新版本可能存在冲突
yarn add stylus@0.54.8 stylus-loader@3.0.2 -D
```
